# CS50
CS50 是哈佛大学的计算机导论公开课，这门课每年都会录制更新一个版本，并公开在网络上。课程对许多计算机体系的细节解释的特别到位，直戳核心编程思想，非常有助于我们对计算机体系建立基本的 Overview。
